# Phishing URL Detector

A modern AI-inspired phishing URL detection web application built using Python, Flask, JavaScript, HTML, and CSS.

The system analyzes suspicious URLs using heuristic threat detection techniques and provides a modern cybersecurity-focused interface with real-time scanning animations and threat visualization.

---

## Features

* Real-time phishing URL analysis
* Interactive cybersecurity dashboard UI
* Intelligent heuristic-based threat detection
* Animated threat scanning experience
* Dynamic threat level meter
* Responsive modern design
* Interactive frontend with JavaScript
* Glassmorphism-inspired UI styling

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Phishing-URL-Detector.git
```

Navigate into the project folder:

```bash
cd Advanced-Phishing-URL-Detector
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```

---

## Detection Logic

The system performs heuristic-based URL analysis using:

* URL length analysis
* HTTPS security validation
* Suspicious keyword detection
* Threat scoring system
